# Lightning Talk Materials

このディレクトリは、このリポジトリを題材にしたライトニングトーク用の資料置き場です。

## Title

LLMに手伝ってもらってBlender保存時glTF書き出しでライブプレビュー

## Files

- `draft.md`: 発表で実際に話す内容の草稿
- `outline.md`: 時間配分を意識した構成メモ
- `notes.md`: 補足メモ、Q&A、削った話題
- `references/links.md`: 参考リンク
- `assets/`: 図、スクリーンショット、発表用画像

## Suggested Flow

1. `outline.md` で発表の骨子を作る
2. `draft.md` で話す文章に落とす
3. 必要に応じて `assets/` に図を追加する
4. `notes.md` に補足や想定質問を書く
